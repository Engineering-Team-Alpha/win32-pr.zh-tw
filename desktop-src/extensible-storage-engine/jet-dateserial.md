# <a name="jet_dateserial"></a>JET_DATESERIAL


_**適用于：** Windows |Windows Server_

## <a name="jet_dateserial"></a>JET_DATESERIAL

**JET_DATESERIAL** 資料類型代表從100年起算的日期（以小數天為單位）。

```cpp
    typedef double JET_DATESERIAL;
```

### <a name="data-types"></a>資料類型

JET_DATESERIAL

將雙精確度 (8 個位元組) 浮點數。 這與 c + + [日期](https://msdn.microsoft.com/library/82ab7w69(VS.71).aspx) 類型相同。
