# <a name="avencmpvgenerateheaderseqext-property"></a>AVEncMPVGenerateHeaderSeqExt 屬性

指定編碼器是否產生順序延伸標頭。 這個屬性會套用至 MPEG 視頻編碼器。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**變異 \_BOOL** (**VT \_ bool**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncMPVGenerateHeaderSeqExt**

## <a name="remarks"></a>備註

如果值為 **VARIANT \_ TRUE**，編碼器會產生順序延伸標頭。
