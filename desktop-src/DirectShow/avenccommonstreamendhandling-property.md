# <a name="avenccommonstreamendhandling-property"></a>AVEncCommonStreamEndHandling 屬性

指定編碼器是否在資料流程的結尾 (Gop) 捨棄部分的圖片群組。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncCommonStreamEndHandling**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncCommonStreamEndHandling**](/windows/win32/api/codecapi/ne-codecapi-eavenccommonstreamendhandling) 列舉的成員。
