# <a name="external-device-control-interfaces"></a>外部裝置控制介面

這些介面支援對外部裝置（例如 DV 攝影機）的應用程式控制。



| 介面                                            | 描述                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | 控制攝影機。                                                                       |
| [**IAMExtDevice**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | 控制外部裝置，例如 VCR 或攝像機。                                  |
| [**IAMExtTransport**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | 控制 VCR 傳輸功能，例如播放、暫停、錄製、向前快轉和倒轉。 |
| [**IAMTimecodeDisplay**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | 在顯示 SMPTE 時間碼的裝置上設定屬性。                                |
| [**IAMTimecodeGenerator**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | 在產生 SMPTE 時間碼的裝置上設定屬性。                               |
| [**IAMTimecodeReader**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | 從外部裝置讀取 SMPTE 時間碼。                                            |
