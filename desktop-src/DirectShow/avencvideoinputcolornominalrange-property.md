# <a name="avencvideoinputcolornominalrange-property"></a>AVEncVideoInputColorNominalRange 屬性

指定輸入影片的名義範圍。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoInputColorNominalRange**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoColorNominalRange**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolornominalrange) 列舉的成員。
