# <a name="copyright"></a>著作權

**著作權** 屬性包含內容的著作權訊息。

## <a name="global-constant"></a>全域常數

g \_ wszWMCopyright

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="example"></a>範例



| 檔案類型 | 範例值                          |
|-----------|----------------------------------------|
| 任意       | "著作權 (C) Microsoft Corporation）。 |



 

## <a name="remarks"></a>備註

著作權訊息的文字不符合任何標準。 個別公司可以使用任何適當的格式。

這個屬性會套用至整個檔案。
