# <a name="wmtoolversion"></a>WM/ToolVersion

**WM/ToolVersion** 屬性包含用來建立檔案之應用程式的版本號碼。 如果您使用此屬性，您也應該包含使用 [**WM/工具工具**](wm-toolname.md)的應用程式名稱。

## <a name="global-constant"></a>全域常數

g \_ wszWMToolVersion

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**
