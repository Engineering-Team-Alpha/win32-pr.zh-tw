# <a name="wmauthorurl"></a>WM/AuthorURL

**WM/AuthorURL** 屬性包含作者的網站位址。

## <a name="global-constant"></a>全域常數

g \_ wszWMAuthorURL

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

這個屬性所指定的位址應該與檔案中 [**author**](author.md) 屬性的演出者或作者相對應。
