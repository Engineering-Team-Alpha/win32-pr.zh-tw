# <a name="wmmediastationcallsign"></a>WM/MediaStationCallSign

**WM/MediaStationCallSign** 屬性包含廣播內容之工作站的撥號符號。

## <a name="global-constant"></a>全域常數

g \_ wszWMMediaStationCallSign

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

這個屬性會用於廣播電視和無線電。
