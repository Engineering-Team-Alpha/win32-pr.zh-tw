# <a name="bannerimagedata"></a>BannerImageData

**BannerImageData** 屬性包含商標用途的橫幅影像。 這個屬性所包含的影像類型是由 [**BannerImageType**](bannerimagetype.md) 屬性所描述。

## <a name="global-constant"></a>全域常數

g \_ wszWMBannerImageData

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 二進位**

## <a name="remarks"></a>備註

儲存在這個屬性中的影像可以是點陣圖、JPEG 或 GIF 格式。
