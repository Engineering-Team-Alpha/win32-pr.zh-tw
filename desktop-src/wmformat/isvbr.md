# <a name="isvbr"></a>IsVBR

**IsVBR** 屬性會指定資料流程中的內容是否使用變數位元速率進行編碼， (VBR) 編碼。

## <a name="global-constant"></a>全域常數

g \_ wszWMIsVBR

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ BOOL**

## <a name="remarks"></a>備註

存取寫入器物件的 **IWMHeaderInfo3** 介面時，您可以加入或變更此值。 在其他物件 (中繼資料編輯器、讀取器和同步讀取器) ，此值是唯讀的。
