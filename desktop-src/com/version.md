# <a name="version-com"></a>COM)  (版本

指定控制項的版本號碼。

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      Version = value
```

## <a name="remarks"></a>備註

這是 **REG \_ SZ** 值。

版本號碼應符合與控制項相關聯的類型程式庫版本。
