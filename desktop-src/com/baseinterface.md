# <a name="baseinterface"></a>Typeinterface

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Interface
   {IID}
      BaseInterface = name
```

這是 **REG \_ SZ** 值。
