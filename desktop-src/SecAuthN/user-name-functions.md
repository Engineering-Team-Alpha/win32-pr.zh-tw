# <a name="user-name-functions"></a>使用者名稱函式

使用者名稱函數包含單一函式 [**NPGetUser**](/windows/desktop/api/Npapi/nf-npapi-npgetuser)。 此函式會抓取目前預設的使用者名稱，或用來建立網路連線的使用者名稱。
