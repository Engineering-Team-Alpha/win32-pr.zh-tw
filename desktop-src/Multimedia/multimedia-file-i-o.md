# <a name="multimedia-file-io"></a>多媒體檔案 i/o

多媒體檔案 i/o 服務提供的功能比標準作業系統服務更多功能，包括支援緩衝處理的 i/o、資源交換檔案格式 (RIFF) 檔案、記憶體檔案和自訂儲存體系統。 此外，多媒體檔案 i/o 服務已針對對效能敏感的應用程式進行優化。
