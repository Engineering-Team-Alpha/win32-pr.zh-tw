# <a name="capture-and-audio-drivers"></a>捕獲和音訊驅動程式

捕捉驅動程式和基礎硬體可以決定影片捕獲的數個層面，包括可接受的影片來源、顯示選項、格式和壓縮選項。 音訊驅動程式會指定音訊格式，以及搭配已捕捉音訊資料使用的選擇性壓縮選項。
