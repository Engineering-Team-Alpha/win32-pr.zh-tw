# <a name="about-waveform-audio"></a>關於波形音訊

有幾種方法可以使用波形音訊將音效新增至您的應用程式。 此處所記載最簡單的方法是使用 [**PlaySound**](/previous-versions//dd743680(v=vs.85)) 函數。 大部分的其他波形音訊 API 元素相對較低層級。

如需詳細資訊，請參閱下列主題：

-   [簡易音訊播放](simple-audio-playback.md)
-   [波形-音訊介面](waveform-audio-interface.md)
-   [輔助音訊介面](auxiliary-audio-interface.md)
-   [音訊資料區塊](audio-data-blocks.md)
-   [使用音訊函數處理錯誤](handling-errors-with-audio-functions.md)

除了此處討論的 Api 之外，Media Control 介面還可以用來播放波形音訊以及其他媒體，例如 MIDI 音效和影片。 如需詳細資訊，請參閱 [MCI](mci.md)。
