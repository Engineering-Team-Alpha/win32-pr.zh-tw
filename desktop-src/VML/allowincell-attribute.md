# <a name="allowincell-attribute"></a>AllowInCell 屬性

決定是否可以將圖形放在資料表中。 讀取/寫入 **VgTriState**。

**適用於**

[形狀](shape-element--vml.md)

**標記語法**

v：*element* o:allowincell = "*expression*" >

**備註**

由 Microsoft Word 使用。 預設值為 **[False]** 。

*Microsoft Office Extensions 屬性*
