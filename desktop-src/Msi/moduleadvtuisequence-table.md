# <a name="moduleadvtuisequence-table"></a>ModuleAdvtUISequence 資料表

請勿撰寫此資料表。 Windows Installer 不會使用 [AdvtUISequence 資料表](advtuisequence-table.md)。 AdvtUISequence 資料表不應該存在，也不應該留在安裝資料庫中。 也不要編寫 ModuleAdvtUISequence 資料表。
