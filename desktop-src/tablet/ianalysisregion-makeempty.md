# <a name="ianalysisregionmakeempty-method"></a>IAnalysisRegion：： MakeEmpty 方法

減少 [**IAnalysisRegion**](ianalysisregion.md) 以代表空白區域。

## <a name="syntax"></a>語法


```C++
HRESULT MakeEmpty();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

如需傳回值的描述，請參閱 [類別和介面-筆跡分析](classes-and-interfaces---ink-analysis.md)。
