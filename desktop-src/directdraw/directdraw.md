# <a name="directdraw"></a>DirectDraw

本章節包含使用 (API) 的 DirectX 應用程式開發介面的 DirectDraw 元件進行程式設計的相關資訊。

不再建議使用 DirectDraw。 使用 Direct3D 9.0 的版本，所有的二維功能都包含在 Direct3D 中、其在 D3DX 中的相關 helper 函式，以及 DirectX 11 技術 Direct2D。 不過，此節仍提供 DirectDraw 參考檔。
