# <a name="system-errors-in-active-directory-domain-services"></a>Active Directory Domain Services 中的系統錯誤

本主題包含 Active Directory Domain Services 的函式中的系統錯誤傳回值。



| 值                             | 描述                                                           |
|-----------------------------------|-----------------------------------------------------------------------|
| **hrFileClose**<br/>        | 無法關閉 DOS 檔案。<br/>                              |
| **hrOutOfThreads**<br/>     | 無法啟動執行緒，因為沒有任何可用的執行緒。<br/> |
| **hrTooManyIO**<br/>        | 系統忙碌中，因為 i/o 太多。<br/>        |
| **hrBFNotSynchronous**<br/> | 緩衝區頁面已收回。<br/>                          |
| **hrBFPageNotFound**<br/>   | 找不到此頁面。<br/>                                   |
| **hrBFInUse**<br/>          | 無法放棄緩衝區。<br/>                              |
