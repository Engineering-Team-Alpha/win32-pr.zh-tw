# <a name="basicdevicemodelname-property"></a>BasicDevice. ModelName 屬性

取得裝置的模型名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_ModelName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>屬性值

裝置模型名稱的指標。
