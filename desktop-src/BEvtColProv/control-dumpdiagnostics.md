# <a name="dumpdiagnostics-method-of-the-control-class"></a>Control 類別的 DumpDiagnostics 方法

將診斷資訊傾印到記錄檔中。

## <a name="syntax"></a>語法


```mof
void DumpDiagnostics();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
