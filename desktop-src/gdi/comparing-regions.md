# <a name="comparing-regions"></a>比較區域

應用程式會比較兩個區域，藉由呼叫 [**EqualRgn**](/windows/desktop/api/Wingdi/nf-wingdi-equalrgn) 函數來判斷它們是否相同。 如果兩個區域的大小和形狀相等，EqualRgn 會將它們視為相同。
