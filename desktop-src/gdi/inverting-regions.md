# <a name="inverting-regions"></a>反轉區域

應用程式會藉由呼叫 [**InvertRgn**](/windows/desktop/api/Wingdi/nf-wingdi-invertrgn) 函式來反轉區域內出現的色彩。 在單色顯示器上， **InvertRgn** 會將白色圖元的黑色和黑色圖元白色。 在色彩畫面上，這項反轉取決於用來產生螢幕色彩的技術類型。
