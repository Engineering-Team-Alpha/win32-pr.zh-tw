# <a name="about-coordinate-spaces-and-transformations"></a>關於座標空間和轉換

下列類型的應用程式會使用座標空間和轉換：

-   桌上型電腦發佈應用程式 (為頁面的「縮放」部分，或在視窗) 中顯示連續的頁面。
-   電腦輔助設計 (CAD) 應用程式 (旋轉物件、調整繪圖或建立) 的觀點。
-   試算表應用程式 (移動和調整圖形大小) 。
