# <a name="painting-regions"></a>繪製區域

應用程式會使用目前在裝置內容中選取的筆刷， [**PaintRgn**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) 函式來填滿區域的內部。 此函式會使用目前的多邊形填滿模式 (替代和繞組的) 。
