# <a name="about-rectangles"></a>關於矩形

矩形用於游標裁剪區域、工作區的無效部分、顯示格式化文字的區域，或捲軸區域。 您的應用程式也可以使用矩形來填滿、框架，或使用指定的筆刷來將部分工作區反轉，以及抓取視窗或視窗工作區的座標。

-   [矩形座標](rectangle-coordinates.md)
-   [矩形作業](rectangle-operations.md)
