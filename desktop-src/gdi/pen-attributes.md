# <a name="pen-attributes"></a>畫筆屬性

有七個畫筆屬性定義畫筆的類型及其特性：寬度、樣式、色彩、模式、影線、結束樣式和聯結樣式。 表面和幾何畫筆都有寬度、樣式和色彩屬性。 只有幾何畫筆具有圖樣、影線、end 樣式和聯結樣式屬性。 模式和選擇性影線屬性通常與筆刷相關聯，但也可以搭配幾何畫筆使用。

如需詳細資訊，請參閱下列主題：

-   [畫筆寬度](pen-width.md)
-   [畫筆樣式](pen-style.md)
-   [畫筆色彩](pen-color.md)
-   [畫筆模式](pen-pattern.md)
-   [畫筆影線](pen-hatch.md)
-   [畫筆結束端點](pen-end-cap.md)
-   [畫筆聯結](pen-join.md)
