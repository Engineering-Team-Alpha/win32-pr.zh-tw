# <a name="paths-windows-gdi"></a>Windows GDI (路徑) 

*路徑* 是一或多個圖形 (或圖形) 填滿、空心或填滿和空心。 路徑用於繪製和繪製應用程式。 電腦輔助設計 (CAD) 應用程式使用路徑來建立唯一的裁剪區域、繪製不規則圖形的輪廓，以及填滿不規則圖形的內部。 不規則圖形是由貝茲曲線和直線組成的圖形。  (一般圖形包括橢圓形、圓形、矩形和多邊形。 )
