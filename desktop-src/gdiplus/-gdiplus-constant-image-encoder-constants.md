# <a name="image-encoder-constants"></a>影像編碼器常數

[**Image 類別的**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-image) [Image：： Save 方法](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-image-save(inistream_inconstclsid_inconstencoderparameters))和 [image：： SaveAdd 方法](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-image-saveadd(inimage_inconstencoderparameters))方法會收到 [**system.drawing.imaging.encoderparameters>**](/windows/win32/api/gdiplusimaging/nl-gdiplusimaging-encoderparameters)物件，其中包含 [**EncoderParameter**](/windows/win32/api/gdiplusimaging/nl-gdiplusimaging-encoderparameter)物件的陣列。 每個 **EncoderParameter** 物件都有指定參數分類的 GUID 資料成員。 下列常數定義于 Gdiplusimaging 中，代表指定各種參數類別目錄的 Guid。

-   EncoderChrominanceTable
-   EncoderColorDepth
-   EncoderColorSpace
-   EncoderCompression
-   EncoderLuminanceTable
-   EncoderQuality
-   EncoderRenderMethod
-   EncoderSaveFlag
-   EncoderScanMethod
-   EncoderTransformation
-   EncoderVersion
-   EncoderImageItems
-   EncoderSaveAsCMYK
