# <a name="about-gdi"></a>關於 GDI +

Windows GDI + 是 Windows XP 作業系統或 Windows Server 2003 作業系統的一部分，提供二維向量圖形、影像處理和印刷樣式。 GDI + 改善 Windows 圖形裝置介面 (GDI)  (舊版 Windows) 隨附的圖形裝置介面，方法是新增功能，並將現有的功能優化。
