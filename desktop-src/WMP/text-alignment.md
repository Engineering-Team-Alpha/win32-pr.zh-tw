# <a name="text-alignment"></a>文字對齊

您必須定義您想要使用之文字顯示方塊的對齊方式。 下表中顯示的值有效。



| 值  | 描述                |
|--------|----------------------------|
| Left   | 文字靠左對齊。  |
| Center | 文字置中。      |
| Right  | 文字是靠右對齊。 |
