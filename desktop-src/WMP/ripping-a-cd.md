# <a name="ripping-a-cd"></a>將 CD 翻錄

有兩種方式可以使用 Windows Media Player 控制項來燒錄 CD。 Windows Media Player 9 可以使用 **IWMPPlayerServices：： setTaskPane** 來燒錄 cd。 Windows Media Player 11 引進 **IWMPCdromRip** 介面來將 cd 翻錄。 此介面提供的功能比 **IWMPPlayerServices：： setTaskPane** 還多，建議您使用 **IWMPCdromRip** （如果有的話）。

下列各節說明如何使用 Windows Media Player 來翻錄 CD。

-   [使用 IWMPCdromRip 介面進行翻錄](ripping-by-using-the-iwmpcdromrip-interface.md)
-   [使用 IWMPPlayerServices：： setTaskPane 進行翻錄](ripping-by-using-iwmpplayerservices--settaskpane.md)
