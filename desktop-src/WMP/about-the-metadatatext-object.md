# <a name="about-the-metadatatext-object"></a>關於 MetadataText 物件

某些中繼資料屬性（例如 **WM/歌詞 \_ 內容**）會提供文字，並伴隨文字描述。 **MetadataText** 物件會提供一種方式來個別取出文字和描述。 當您使用 *媒體* 時。**getItemInfoByType** 若要取得複雜文字屬性的相關資訊，您可以取出具有屬性的 **MetadataText** 物件，其中包含個別資料。
