# <a name="creating-the-volumethumb-file"></a>建立 VolumeThumb 檔案

VolumeThumb 檔案包含將用於磁片區的捲軸的捲動方塊影像。

下圖是 VolumeThumb 檔。

![volumethumb 檔案](images/ceswmvol.png)

這包含兩個 thumb 影像。 左邊的是一般狀態，右側的則是已推送的狀態。 沒有已停用的狀態。 如果停用了 [已停用]，則會從超級檔案顯示已停用的程式背景影像。
