# <a name="skin-definition-file-mobile"></a> (Mobile) 的外觀定義檔

面板定義檔是您將所有元件放在一起的位置。 您可以定義要使用的影像、指定您想要讓使用者看到的使用者介面元素，以及輸入函式、座標、檔案名等等。
