# <a name="item-element"></a>ITEM 元素

**Item** 元素代表清單方塊或快顯控制項中的專案。 **ITEM** 元素一律為 **LISTBOX** 或 **POPUP** 元素的子系。

**ITEM** 元素支援下列屬性。



| 屬性               | 描述                                                                  |
|-------------------------|------------------------------------------------------------------------------|
| [value](item-value.md) | 指定或抓取針對清單方塊或快顯項目目顯示的文字。 |



 

> [!Note]  
> 此元素需要 Windows XP 或更新版本的 Windows Media Player。
