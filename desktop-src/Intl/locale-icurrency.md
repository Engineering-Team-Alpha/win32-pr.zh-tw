# <a name="locale_icurrency"></a>地區設定 \_ ICURRENCY

貨幣符號在正貨幣模式中的位置。



| 值 | 意義                                            |
|-------|----------------------------------------------------|
| 0     | 前置詞，無分隔，例如 $1。1           |
| 1     | 尾碼，沒有分隔，例如 $1。1           |
| 2     | 前置詞，1-字元分隔，例如 $1。1 |
| 3     | 尾碼，1個字元的分隔，例如 $1。1 |
