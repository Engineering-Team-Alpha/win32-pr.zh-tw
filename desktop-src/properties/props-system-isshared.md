# <a name="systemisshared"></a>System. IsShared

指出專案是否為共用。 這只會檢查非繼承的 Acl。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.IsShared
   shellPKey = PKEY_IsShared
   formatID = EF884C5B-2BFE-41BB-AAE5-76EEDF4F9902
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.IsShared
   shellPKey = PKEY_IsShared
   formatID = EF884C5B-2BFE-41BB-AAE5-76EEDF4F9902
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
