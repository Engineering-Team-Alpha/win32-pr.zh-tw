# <a name="certificate-template-interfaces"></a>憑證範本介面

下列介面可以用來以程式設計方式存取、修改和儲存憑證要求範本。



| 介面                                                                    | 描述                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IX509CertificateTemplate**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplate)                 | 代表憑證要求範本。                                                    |
| [**IX509CertificateTemplates**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplates)               | 管理 [**IX509CertificateTemplate**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplate) 物件的集合。 |
| [**IX509CertificateTemplateWritable**](/windows/desktop/api/Certenroll/nn-certenroll-ix509certificatetemplatewritable) | 在範本存放區中加入或刪除範本。                                       |
