# <a name="ui_pkey_automaticcolorlabel"></a>UI \_ PKEY \_ AutomaticColorLabel

識別 UI \_ PKEY \_ AutomaticColorLabel 屬性。

```
propertyDescription
   name = UI_PKEY_AutomaticColorLabel
   shellPKey = UI_PKEY_AutomaticColorLabel
   formatID = 00000406-7363-696e-8441798acf5aebb7
   propID = 406
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY AutomaticColorLabel 來查詢與 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)的 [**自動**] 按鈕相關聯的標籤值。 應用程式會使用這個按鈕，視需要指定預設色彩。

UI \_ PKEY \_ AutomaticColorLabel 只適用于 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) ，其中 `ThemeColors` 或 `StandardColors` 指定為 **ColorTemplate** 屬性的值。
