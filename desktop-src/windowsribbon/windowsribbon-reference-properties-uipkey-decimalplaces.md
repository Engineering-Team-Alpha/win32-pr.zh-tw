# <a name="ui_pkey_decimalplaces"></a>UI \_ PKEY \_ 小數

識別 UI \_ PKEY \_ 小數屬性。

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY 小數值來查詢[微調](windowsribbon-controls-spinner.md)控制項之 [編輯] 欄位中的小數點後所顯示的位數。
