# <a name="ui_pkey_largeimage"></a>UI \_ PKEY \_ LargeImage

識別 UI \_ PKEY \_ LargeImage 屬性。

```
propertyDescription
   name = UI_PKEY_LargeImage
   shellPKey = UI_PKEY_LargeImage
   formatID = 00000007-7363-696e-8441798acf5aebb7
   propID = 7
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY LargeImage 來查詢與功能區控制項相關聯的大型影像。

屬性值是 [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) 物件。
