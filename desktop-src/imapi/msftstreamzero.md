# <a name="msftstreamzero-object"></a>MsftStreamZero 物件

您可以使用這個物件來建立唯讀資料資料流程，其資料會初始化為零。 您必須呼叫 **SetSize** 方法來設定所要求的資料流程大小。

## <a name="members"></a>成員

**MsftStreamZero** 物件未定義任何成員。
