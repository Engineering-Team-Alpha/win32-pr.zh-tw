## <a name="resource-views"></a><span data-ttu-id="601ba-137">資源檢視</span><span class="sxs-lookup"><span data-stu-id="601ba-137">Resource Views</span></span>

<span data-ttu-id="601ba-141">View 可以搭配無別別資源使用。</span><span class="sxs-lookup"><span data-stu-id="601ba-141">A view can be used with a typeless resource.</span></span> <span data-ttu-id="601ba-142">也就是說，您可以在編譯時期建立資源，並在資源系結至管線時宣告資料類型。</span><span class="sxs-lookup"><span data-stu-id="601ba-142">That is, you can create a resource at compile time and declare the data type when the resource is bound to the pipeline.</span></span> <span data-ttu-id="601ba-143">針對無值資源建立的視圖一律會有相同數目的每個元件的位數;資料的轉譯方式取決於指定的格式。</span><span class="sxs-lookup"><span data-stu-id="601ba-143">A view created for a typeless resource always has the same number of bits per component; the way the data is interpreted is dependent on the format specified.</span></span> <span data-ttu-id="601ba-144">指定的格式必須與建立資源時所使用的無格式的系列相同。</span><span class="sxs-lookup"><span data-stu-id="601ba-144">The format specified must be from the same family as the typeless format used when creating the resource.</span></span> <span data-ttu-id="601ba-145">例如，使用 R8G8B8A8 無型別格式建立的資源 \_ 無法視為 R32 \_ FLOAT 資源，即使這兩種格式在記憶體中的大小可能相同也一樣。</span><span class="sxs-lookup"><span data-stu-id="601ba-145">For example, a resource created with the R8G8B8A8\_TYPELESS format cannot be viewed as a R32\_FLOAT resource even though both formats may be the same size in memory.</span></span>
