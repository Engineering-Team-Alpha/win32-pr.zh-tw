# <a name="tiled-resource-cross-process-and-device-sharing"></a>磚資源跨進程和裝置共用

磚集區可以與其他處理序共用，就像傳統資源一樣。 參考磚集區的並排資源無法跨裝置和進程共用。 但是，個別的進程可以建立自己的磚資源，對應至這些已並排資源之間共用的磚集區。

共用的磚集區無法調整大小。
