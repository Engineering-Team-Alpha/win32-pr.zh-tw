# <a name="windows-networking-wnet"></a>Windows 網路 (WNet) 

Windows 網路 (WNet) 函式可讓您在應用程式中執行網路功能，而不需要針對特定的網路提供者或實體網路執行進行額度。 這是因為 WNet 函數與網路無關。
