# <a name="windows-networking-callbacks"></a>Windows 網路回呼

下列函式原型是在 Windows 網路功能中使用。



| 函式原型                                 | Description                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | 取得使用者設定檔路徑的函式原型。                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | 傳回與特定 Windows 進程相關聯之使用者帳戶原則資訊的函式原型。 |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | 將本機設定檔與對應的全域設定檔協調的函式原型。                        |
