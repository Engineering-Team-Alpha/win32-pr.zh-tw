# <a name="swbemeventsourcesecurity_-property"></a>SWbemEventSource 安全性 \_ 屬性

[**SWbemEventSource**](swbemeventsource.md)物件的 **security \_** 屬性會取得或設定 **SWbemEventSource** 物件的安全性設定。 這個屬性是 [**SWbemSecurity**](swbemsecurity.md) 物件。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemEventSource.Security_ As Object
```
