# <a name="generic-com-error-codes"></a>一般 COM 錯誤碼

泛型 COM 錯誤碼表示所有 COM 模組所產生的作業狀態。 下表包含任何 Win32 平臺上的一些錯誤碼範例。



| 錯誤碼     | 十六進位值  | Description              |
|----------------|------------|--------------------------|
| E \_ 中止       | 0x80004004 | 作業已中止。       |
| E \_ 失敗        | 0x80004005 | 未指定的錯誤。       |
| E \_ NOINTERFACE | 0x80004002 | 不支援的介面。 |
| E \_ >NOTIMPL     | 0x80004001 | 未實作。         |
| E \_ 指標     | 且顯示0x80004003 | 指標無效。         |
| E 未 \_ 預期  | 0x8000FFFF | 嚴重失敗。    |
