| 方法                                   | 描述                                                                                              |
|------------------------------------------|----------------------------------------------------------------------------------------------------------|
| **CCoreADsObject::InitializeCoreObject** | 初始化這個物件的基本 [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) 屬性，這是建立物件的便利性。 |
| **CCoreADsObject::CCoreADsObject**       | 標準建立者。                                                                                        |
| **CCoreADsObject：： ~ CCoreADsObject**      | 標準的函式。                                                                                     |
