# <a name="adsi-router"></a>ADSI 路由器

ADSI 路由器位於用戶端和 ADSI 提供者之間。 它會執行一組核心物件，這些物件提供所有提供者通用的基本功能和服務。 因此，任何提供者也都支援路由器支援的任何功能，除非提供者選擇其他。 ADSI 路由器模組對應于 Activeds.dll。
