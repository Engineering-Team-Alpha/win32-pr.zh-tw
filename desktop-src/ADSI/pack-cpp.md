# <a name="packcpp"></a>包。Cpp

在範例提供者元件中，會在 pack 中找到封裝和解壓縮資料類型的程式碼範例。 當屬性已載入或從屬性快取中移除時，會使用下列函式：

-   **PackStringinVariant**
-   **UnpackStringinVariant**
-   **PackLONGinVariant**
-   **UnpackLONGfromVariant**
-   **PackDATEinVariant**
-   **UnpackDATEinVariant**
-   **PackVARIANT \_ BOOLinVariant**
-   **UnpackVARIANT \_ BOOLfromVariant**
-   **PackVARIANTinVariant**
-   **UnpackVARIANTfromVariant**
