# <a name="bluetooth-and-getsockname"></a>藍牙和 getsockname

藍牙使用 [**getsockname**](/windows/desktop/api/winsock/nf-winsock-getsockname) 函式，透過 [**先前的系**](/windows/desktop/api/winsock/nf-winsock-bind) 結函式呼叫，來取得配置給通訊端的伺服器裝置位址和埠號碼。

如需詳細資訊，請參閱 [Windows 通訊端](/windows/desktop/WinSock/windows-sockets-start-page-2)。
