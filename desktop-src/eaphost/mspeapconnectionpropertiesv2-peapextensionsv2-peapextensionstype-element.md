# <a name="peapextensionsv2-peapextensionstype-element-connection-properties-v1"></a>PeapExtensionsV2 (PeapExtensionsType) 元素 (連接屬性 v1) 

**PeapExtensionsV2 (PeapExtensionsType)** 專案可讓架構的未來增強功能。

``` syntax
<xs:element name="PeapExtensionsV2"
    type="PeapExtensionsTypeV2"
 />
```

**PeapExtensionsV2** 元素是由 [**PeapExtensionsType**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md)元素定義。

## <a name="remarks"></a>備註

**PeapExtensionsV2** 元素是選擇性的。
