# <a name="mf_pd_total_file_size-attribute"></a>MF \_ PD \_ TOTAL \_ FILE \_ SIZE 屬性

指定原始程式檔的大小總計（以位元組為單位）。 這個屬性會套用至展示描述項。 媒體來源可以選擇性地設定這個屬性。

## <a name="data-type"></a>資料類型

**UINT64**

## <a name="remarks"></a>備註

這個屬性的 GUID 常數是從 mfuuid 匯出。
