# <a name="imfsourcebuffernotifyonabort-method"></a>IMFSourceBufferNotify：： OnAbort 方法

用來表示已中止來源緩衝區。

## <a name="syntax"></a>語法


```C++
void OnAbort();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
