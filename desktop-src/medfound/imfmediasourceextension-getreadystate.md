# <a name="imfmediasourceextensiongetreadystate-method"></a>IMFMediaSourceExtension：： GetReadyState 方法

取得媒體來源的就緒狀態。

## <a name="syntax"></a>語法


```C++
MF_MSE_READY GetReadyState();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值
