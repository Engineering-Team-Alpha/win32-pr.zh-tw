# <a name="create-time-stamp-attribute"></a>建立時間戳記屬性

此物件的建立日期。 此值為已複寫。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------|
| CN                | 建立時間戳記                                             |
| Ldap-顯示名稱 | createTimeStamp                                               |
| 大小              | 8 個位元組                                                       |
| 更新許可權  | 此值是由系統所設定。                              |
| 更新頻率  | 建立物件時。                                   |
| Attribute-Id      | 2.5.18.1                                                      |
| 系統識別碼-Guid    | 2df90d73-009f-11d2-aa4c-00c04fd7d83a                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
