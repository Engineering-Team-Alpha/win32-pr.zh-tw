# <a name="ms-pki-oid-localizedname-attribute"></a>ms-PKI-OID-LocalizedName 屬性

包含用來依地區設定描述 OID 的顯示名稱清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-PKI-OID-LocalizedName                    |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-OIDLocalizedName                      |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1712                     |
| 系統識別碼-Guid    | 7d59a816-bb05-4a72-971f-5c1331f67559        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
