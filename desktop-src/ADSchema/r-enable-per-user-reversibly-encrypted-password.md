# <a name="enable-per-user-reversibly-encrypted-password-extended-right"></a>啟用每個使用者可反向加密-密碼延伸許可權

擴充控制存取權限，可讓使用者啟用或停用使用者和電腦物件的「可還原的加密密碼」設定。



| 進入 | 值 |
|--------------|-----------------------------------------------|
| CN           | 啟用每位使用者可反向加密-密碼 |
| 顯示名稱 | 啟用每個使用者的可反向加密密碼 |
| 許可權-GUID  | 05c74c5e-4deb-43b4-bd9f-86664c2a7fd5          |
