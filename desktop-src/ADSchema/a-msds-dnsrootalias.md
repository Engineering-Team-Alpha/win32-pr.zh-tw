# <a name="ms-ds-dnsrootalias-attribute"></a>DnsRootAlias 屬性

用來儲存網域別名。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | DnsRootAlias                          |
| Ldap-顯示名稱 | DnsRootAlias                           |
| 大小              | 大小上限為255個位元組。                     |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 只有在網域重建時。             |
| Attribute-Id      | 1.2.840.113556.1.4.1719                     |
| 系統識別碼-Guid    | 2143acca-eead-4d29-b591-85fa49ce9173        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
