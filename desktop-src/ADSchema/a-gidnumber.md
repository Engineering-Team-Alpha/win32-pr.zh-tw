# <a name="gidnumber-attribute"></a>gidNumber 屬性

包含可唯一識別系統管理網域中群組的整數值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | gidNumber                            |
| Ldap-顯示名稱 | gidNumber                            |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.1                      |
| 系統識別碼-Guid    | c5b95f0c-ec9e-41c4-849c-b46597ed6696 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
