# <a name="migrate-sid-history-extended-right"></a>遷移-SID-歷程記錄擴充許可權

擴充許可權，可讓使用者在沒有系統管理員許可權的情況下遷移 SID-History。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 遷移-SID-歷程記錄                  |
| 顯示名稱 | 移轉 SID 歷程記錄                  |
| 許可權-GUID  | ba33815a-4f93-4c76-87f3-57574bff8109 |
