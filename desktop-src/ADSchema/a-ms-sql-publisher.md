# <a name="ms-sql-publisher-attribute"></a>TRANSACT-SQL-發行者屬性

用於複寫的發行者資料庫名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | TRANSACT-SQL-發行者                            |
| Ldap-顯示名稱 | TRANSACT-SQL-發行者                            |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | 當複寫設定時。                  |
| Attribute-Id      | 1.2.840.113556.1.4.1402                     |
| 系統識別碼-Guid    | c1676858-d34b-11d2-999a-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
