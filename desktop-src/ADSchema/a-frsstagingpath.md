|-------------------|---------------------------------------------|
| CN                | FRS-暫存路徑                            |
| Ldap-顯示名稱 | fRSStagingPath                              |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.488                      |
| 系統識別碼-Guid    | 1be8f175-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
