# <a name="certificate-templates-attribute"></a>Certificate-Templates 屬性

包含憑證伺服器所發出之憑證的資訊。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Certificate-Templates                       |
| Ldap-顯示名稱 | 右鍵                        |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.823                      |
| 系統識別碼-Guid    | 2a39c5b1-8960-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
