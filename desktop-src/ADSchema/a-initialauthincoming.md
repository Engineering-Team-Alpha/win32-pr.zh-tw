# <a name="initial-auth-incoming-attribute"></a>初始驗證-傳入屬性

包含用戶端對這部伺服器的初始連入驗證要求的相關資訊。 此伺服器會接著將此要求傳送到網域的驗證服務器。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 初始-驗證-傳入                       |
| Ldap-顯示名稱 | initialAuthIncoming                         |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.539                      |
| 系統識別碼-Guid    | 52458023-ca6a-11d0-afff-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
