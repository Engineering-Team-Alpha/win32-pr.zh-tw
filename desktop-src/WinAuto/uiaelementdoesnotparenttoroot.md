# <a name="uiaelementdoesnotparenttoroot"></a>UiaElementDoesNotParentToRoot

## <a name="text"></a>Text

元素的父鏈未移至根項目

## <a name="type"></a>類型

警告

## <a name="description"></a>描述

從指定元素到 UI 樹狀結構根目錄的導覽已中斷。

此問題可能會導致嘗試流覽 UI 樹狀結構的應用程式產生無法預期的結果。

## <a name="possible-causes"></a>可能的原因

UI 樹狀結構父子式階層的執行不正確。
