# <a name="treetoodeep"></a>TreeTooDeep

## <a name="text"></a>Text

樹狀結構是 {0} 層級深度。

## <a name="type"></a>類型

警告

## <a name="description"></a>描述

元素樹狀結構可能太深。

這個問題會讓依賴螢幕讀取器和鍵盤進行導覽的人員遇到問題，因為進行中的元素似乎需要很長的時間，而且可能會出現迴圈。

## <a name="possible-causes"></a>可能的原因

-   多個元素或其父代是未正確執行 tab 鍵的自訂控制項。 例如，[MSAA 狀態] 屬性未正確更新。
-   應用程式 UI 過於複雜。
