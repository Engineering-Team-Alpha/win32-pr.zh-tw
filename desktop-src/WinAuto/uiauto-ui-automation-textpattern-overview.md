# <a name="ui-automation-support-for-textual-content"></a>消費者介面自動化文字內容的支援

本節說明如何使用 Microsoft 消費者介面自動化，在消費者介面自動化支援的平臺中公開文字控制項的文字內容，包括格式和樣式屬性。 文字控制項包括（但不限於）檔、工具提示、編輯控制項和 rich edit 控制項。
