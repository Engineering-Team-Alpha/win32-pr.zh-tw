# <a name="deviceinfoport-property"></a>DeviceInfo. Port 屬性

抓取 Windows 映像取得 (WIA) 硬體裝置所連線的埠。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = DeviceInfo.Port
```



## <a name="property-value"></a>屬性值

接收埠的字串。
