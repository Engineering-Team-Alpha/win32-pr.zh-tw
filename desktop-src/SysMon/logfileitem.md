# <a name="logfileitem-object"></a>LogFileItem 物件

您可以使用這個類別來取出系統監視器控制項所存取的記錄檔路徑。

若要取出這個物件，請通話記錄檔 [**。**](systemmonitor-logfiles-item.md)

## <a name="members"></a>成員

**LogFileItem** 物件具有下列類型的成員：

-   [屬性](#properties)

### <a name="properties"></a>屬性

**LogFileItem** 物件具有這些屬性。



| 屬性                                    | 描述                                    |
|:--------------------------------------------|:-----------------------------------------------|
| [**路徑**](logfileitem-path.md)<br/> | 抓取記錄檔的路徑。<br/> |
