# <a name="systemmonitortimebarcolor-property"></a>SystemMonitor. TimeBarColor 屬性

抓取或設定時間列的色彩。 時間軸是在圖形視窗間移動的垂直橫條，表示折線圖圖中每個取樣間隔的分佈。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property TimeBarColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>屬性值

時間列的色彩。
