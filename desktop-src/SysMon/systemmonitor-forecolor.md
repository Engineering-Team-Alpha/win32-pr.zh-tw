# <a name="systemmonitorforecolor-property"></a>SystemMonitor 前景屬性

抓取或設定顯示在控制項中的文字色彩。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property ForeColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>屬性值

控制項中顯示的文字色彩

## <a name="remarks"></a>備註

這是環境屬性。 這個屬性的值是由容器所決定。 設定這個屬性的值可能會影響控制項和容器的假像成為單一應用程式。
