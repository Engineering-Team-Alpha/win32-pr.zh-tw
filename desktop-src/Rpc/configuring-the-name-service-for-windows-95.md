# <a name="configuring-the-name-service-for-windows-95"></a>設定 Windows 95 的名稱服務

Windows 95 不使用 Microsoft 定位程式。 若要在 Windows 95 應用程式中使用名稱服務，具有 Windows 95 的電腦必須執行下列其中一項：

-   屬於工作組或網域的一部分，其中包含執行 Windows 2000 或 Windows NT 的電腦，以作為 proxy 名稱服務提供者。
-   連接到執行 NSI daemon 的主機電腦 (nsid) ，其可做為數位設備公司 DCE Cell Directory 服務的閘道。
