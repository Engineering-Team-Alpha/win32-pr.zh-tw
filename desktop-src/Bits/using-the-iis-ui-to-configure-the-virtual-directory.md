# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>使用 IIS UI 來設定虛擬目錄

若要使用 IIS 使用者介面來設定用於 BITS 上傳的虛擬目錄，請從虛擬目錄的內容功能表中選取 [ **屬性** ]。 按 [ **Bits 延伸** 模組] 索引標籤以存取 bits 內容頁面 (必須安裝) 的 bits 伺服器延伸模組 ISAPI。 適當地變更預設屬性值，並啟用上傳的目錄;在目錄啟用之前，您無法上傳檔案。 如需設定屬性值的詳細資訊，請參閱線上說明。
