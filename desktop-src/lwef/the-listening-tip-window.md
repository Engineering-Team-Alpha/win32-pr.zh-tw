# <a name="the-listening-tip-window"></a>接聽提示視窗

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

如果已啟用語音，當使用者按下推播按鍵以開始語音輸入時，就會出現特殊的工具提示視窗。 接聽提示會顯示與 Microsoft Agent 目前輸入狀態相關的內容資訊。 如果相容的語音辨識尚未安裝或已停用，則不會出現接聽提示。
