# <a name="win32-switch"></a>/win32 參數

**/Win32** 參數會指示 MIDL 編譯器產生32位環境的存根檔或類型程式庫檔案。

``` syntax
midl /win32
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="remarks"></a>備註

**/Win32** 參數相當於將 [**/env**](-env.md)參數設定為 win32。
