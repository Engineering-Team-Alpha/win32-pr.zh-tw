# <a name="help--switch"></a>/help (/？ ) 參數

**/Help** (**/？**) 參數會指示編譯器顯示使用訊息，其中詳細說明所有可用的 MIDL 命令列參數和選項。

[**/Confirm**](-confirm.md)參數會顯示使用者選取的 MIDL 編譯器切換設定。

``` syntax
midl /help
midl /?
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="examples"></a>範例

**midl/help**

**midl/？**
