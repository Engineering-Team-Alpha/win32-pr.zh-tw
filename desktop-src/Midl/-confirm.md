# <a name="confirm-switch"></a>/confirm 參數

**/Confirm** 參數會指示編譯器顯示所有 MIDL 編譯器選項，而不處理輸入的 IDL 和選擇性的 ACF 檔。

``` syntax
midl /confirm
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="examples"></a>範例

**midl/confirm**

**midl/confirm @response.rsp 檔案名 .idl**
