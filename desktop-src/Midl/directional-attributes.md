# <a name="directional-attributes"></a>方向屬性

將方向屬性套用至參數，以指出資料的傳遞方向。 這兩個屬性都可以套用至單一參數，但是這樣做會增加與呼叫相關聯的額外負荷。



| 屬性              | 使用方式                                                                 |
|------------------------|-----------------------------------------------------------------------|
| [**在**](in.md)       | 參數正在從呼叫端傳遞至被呼叫的函式。 |
| [**擴展**](out-idl.md) | 參數正在從呼叫的函式傳遞給呼叫端。 |
