# <a name="tb_getdisabledimagelist-message"></a>TB \_ GETDISABLEDIMAGELIST 訊息

抓取工具列控制項用來顯示非作用中按鈕的影像清單。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回非使用中影像清單的控制碼，如果未設定任何非使用中的影像清單，則傳回 **Null** 。
