# <a name="tb_getbuttonsize-message"></a>TB \_ GETBUTTONSIZE 訊息

抓取工具列按鈕目前的寬度和高度（以圖元為單位）。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回 **DWORD** 值，其中包含低字和高單字中的寬度和高度值。
