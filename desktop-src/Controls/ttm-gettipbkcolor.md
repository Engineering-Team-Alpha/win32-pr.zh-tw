# <a name="ttm_gettipbkcolor-message"></a>TTM \_ GETTIPBKCOLOR 訊息

抓取工具提示視窗中的背景色彩。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回表示背景色彩的 **COLORREF** 值。
