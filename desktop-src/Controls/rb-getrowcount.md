# <a name="rb_getrowcount-message"></a>RB \_ GETROWCOUNT 訊息

抓取 Rebar 控制項中的資料列數。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回代表控制項中的帶狀列數目的 **UINT** 值。
