# <a name="custom-draw"></a>自訂繪圖

自訂繪製不是通用控制項;它是許多常見控制項所提供的服務。 自訂繪圖服務可讓應用程式更有彈性地自訂控制項的外觀。 您的應用程式可以利用自訂繪製通知，輕鬆地變更用來顯示專案或手動繪製專案的字型，而不需要進行完整的擁有者繪製。

本章節包含與自訂繪圖搭配使用之程式設計項目的相關資訊。
