# <a name="using-http-server-api"></a>使用 HTTP 伺服器 API

應用程式原始程式檔包含 Http .h 標頭檔，以存取 HTTP 伺服器 API 的函式原型和結構定義。 開發人員可以使用 Httpapi.dll .lib 程式庫檔案來建立使用 HTTP 伺服器 API 的應用程式。 應用程式會在執行時間連結至 Httpapi.dll。
