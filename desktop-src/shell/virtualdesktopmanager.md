# <a name="virtualdesktopmanager-class"></a>VirtualDesktopManager 類別

**VirtualDesktopManager** 會實 [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager) 介面。

## <a name="remarks"></a>備註

您可以使用此介面來與虛擬桌面互動，以及控制程式視窗的啟動位置。 如需詳細資訊，請參閱 [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager)。
