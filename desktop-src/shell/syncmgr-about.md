# <a name="about-synchronization-manager"></a>關於同步處理管理員

同步處理管理員提供集中式的標準技術，可同步處理檔案以在本機電腦上離線使用。 使用者可以從網路儲存體更新所需的檔案，以準備離線使用的電腦。 然後，當電腦重新上線時，任何離線變更都可以複製回網路。 SyncMgr 提供一種通用的使用者介面，讓所有應用程式都可以共用這些使用者介面來同步處理其檔案。

檔案會與通訊協定無關。 例如，電子郵件程式可以使用 SMTP、NMTP 或 POP3 傳送訊息，而瀏覽器可以使用 HTTP，而資料庫可以使用 RPC。 此通訊協定對 SyncMgr 而言是透明的。

SyncMgr 也與儲存體無關，而且可以明確地傳輸檔案、電子郵件訊息、HTML 頁面或資料庫資訊。
