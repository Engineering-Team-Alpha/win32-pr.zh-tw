# <a name="application-scenarios"></a>應用程式情節

可以使用同步處理管理員的應用程式和服務包括下列各項：

-   Microsoft Office 需要準備檔案以供離線使用的應用程式。
-   用戶端快取，可讓您在本機快取檔案案。
-   可以在本機快取 HTML 網頁的瀏覽器。
-   可以在本機快取電子郵件訊息的郵件程式。
-   可以在本機儲存資訊的資料庫。
