# <a name="about-the-dns-wmi-provider"></a>關於 DNS WMI 提供者

DNS 伺服器會將 IP 位址對應到電腦名稱稱，並將電腦名稱稱對應至 IP 位址。 如此一來，就能提供找出網路資源的機制。 DNS WMI 提供者可讓應用程式透過 Windows Management Instrumentation (WMI) 的整合管理架構，與 DNS 伺服器互動。
