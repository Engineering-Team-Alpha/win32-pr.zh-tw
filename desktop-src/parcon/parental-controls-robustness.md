# <a name="parental-controls-robustness"></a>家長監護

請注意，有關強制以身分識別為基礎之登入的社交工程問題，以及限制設計的實際限制，通常會使安全性解決方案免于受到穩固的保護，以確保 outwit 控制項的子系。 Microsoft 的原則是，我們無法保護讓刻意刻意努力規避限制的小孩。 我們可以協助防止非預期或 unsophisticated 的蓄意存取。 請勿在家長監護解決方案中設定家長和監護人的期望。
