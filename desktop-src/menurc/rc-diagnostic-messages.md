# <a name="rc-diagnostic-messages"></a>RC 診斷訊息

本節說明 RC 產生的診斷訊息。 當 RC 無法正確編譯資源時，會出現許多這些訊息。 描述會清楚說明每個訊息的原因。

在訊息描述開頭的括弧中， (V) 的資本 V，表示只有在使用 **/v** (verbose) 選項執行 RC 時，才會顯示訊息。 這些訊息通常是有資訊的，不一定表示錯誤。

這些訊息會依照下列主題中的字母順序列出：

[A](a.md) [B](b.md) [C](c.md) [E](e.md) [F](f.md) [I](i.md) [N](n.md) [O](o.md) [R](r.md) [T](t.md) [U](u.md) [V](v.md) [W](w.md)
