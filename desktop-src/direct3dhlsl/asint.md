# <a name="asint-function"></a>asint 函式

將輸入值的位模式解釋為整數。 如需 **asint** 內建函式的詳細資訊，請參閱 [**ASINT (DirectX HLSL)**](dx-graphics-hlsl-asint.md)。

## <a name="syntax"></a>語法

``` syntax
int asint(
  in  value
);
```

## <a name="parameters"></a>參數

<dl> <dt>

*值* \[在\]
</dt> <dd>

輸入值。

</dd> </dl>

## <a name="return-value"></a>傳回值

類型： **int**

轉譯為整數的輸入。
