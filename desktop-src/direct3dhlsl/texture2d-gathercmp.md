# <a name="texture2dgathercmp-methods"></a>Texture2D：： GatherCmp 方法

針對要在雙線性篩選作業中使用之 [**Texture2D**](sm5-object-texture2d.md) 的四個材質值，會傳回其與比較值的比較。

如需描述基礎 DXBC 指令的詳細資訊，請參閱 [gather4_c](./gather4-c--sm5---asm-.md) 的相關檔。

### <a name="overload-list"></a>多載清單



| 方法                                                                             | 描述                                                                                                      |
|:-----------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**GatherCmp (S、float、float、int)**](sm5-object-texture2d-gathercmp.md)             | 範例並比較材質，並傳回所有四個元件。<br/>                                       |
| [**GatherCmp (S、float、float、int、uint)**](t2d-gathercmp-s-float-float-int-uint-.md) | 範例並比較材質，並傳回所有四個元件以及作業的狀態。<br/> |
