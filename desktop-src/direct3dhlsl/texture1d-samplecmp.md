# <a name="texture1dsamplecmp-methods"></a>Texture1D：： SampleCmp 方法

| 方法                                                                                         | 描述                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp (S、float、float、int)**](dx-graphics-hlsl-to-samplecmp.md)                          | 使用比較值來拒絕範例，以取樣材質。<br/>                                                                                                             |
| [**SampleCmp (S、float、float、int、float)**](t1d-samplecmp-s-float-float-int-float-.md)           | 使用比較值來拒絕範例，並使用選擇性的值來將範例層級 (」 LOD) 值設為，以將材質取樣。<br/>                                     |
| [**SampleCmp (S、float、float、int、float、uint)**](t1d-samplecmp-s-float-float-int-float-uint-.md) | 使用比較值來拒絕範例，並使用選擇性的值來將範例層級 (」 LOD) 值設為，以將材質取樣。 傳回操作的相關狀態。<br/> |
