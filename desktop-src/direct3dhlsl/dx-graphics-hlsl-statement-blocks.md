# <a name="statement-blocks"></a>語句區塊

語句區塊包含一或多個語句;每個語句都是由 [運算式](dx-graphics-hlsl-expressions.md) 和 [運算子](dx-graphics-hlsl-operators.md)組成。


```
{
   statement 1;
   statement 2;
   ...
   statement n;
}
```



語句區塊也會指出子範圍。 語句區塊內宣告的變數只能在區塊內辨識。
