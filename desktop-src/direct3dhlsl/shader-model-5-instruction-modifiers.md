# <a name="shader-model-5-instruction-modifiers"></a>著色器模型5指令修飾詞

指令修飾詞會在將指令寫入註冊之前，先影響指令的結果。 著色器模型5支援下列指令修飾詞。

## <a name="source-operand-modifiers"></a>來源運算元修飾詞

-   [絕對值](absolute-value.md)
-   [否定](negate.md)

## <a name="instruction-result-modifiers"></a>指令結果修飾詞

-   [精確](precise.md)
-   [飽和](saturate.md)
