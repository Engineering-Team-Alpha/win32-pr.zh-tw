# <a name="log2"></a>log2

傳回指定值的以2為底數的對數。



| *ret* log2 (*x*)  |
|-----------------|



 

## <a name="parameters"></a>參數



| 項目                                                   | 描述                            |
|--------------------------------------------------------|----------------------------------------|
| <span id="x"></span><span id="X"></span>*X*<br/> | \[在 \] 指定的值中。<br/> |



 

## <a name="return-value"></a>傳回值

*X* 參數的以2為底數的對數。 如果 *x* 參數為負值，則此函式會傳回不定。 如果 *x* 參數為0，則此函式會傳回 + INF。
