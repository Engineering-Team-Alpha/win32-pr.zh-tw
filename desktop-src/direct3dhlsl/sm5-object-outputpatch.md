# <a name="outputpatch"></a>OutputPatch

代表輸出控制點的陣列，這些點可供輪廓著色器的 patch 常數函式和網域著色器使用。



| 方法                                                       | 描述                         |
|--------------------------------------------------------------|-------------------------------------|
| [**運算子\[\]**](sm5-object-outputpatch-operatorindex.md) | 取得唯讀的資源變數。 |



 

此外，InputPatch 類別支援下列屬性：



| 屬性 | 類型 | Description                   |
|------------|------|-------------------------------|
| 長度     | uint | 控制點的數目。 |



 

## <a name="minimum-shader-model"></a>最小著色器模型

下列著色器模型支援此物件。



| 著色器模型                                                                | 支援 |
|-----------------------------------------------------------------------------|-----------|
| [著色器模型 5](d3d11-graphics-reference-sm5.md) 和更新版本的著色器模型 | 是       |



 

下列著色器類型支援此物件：



| 頂點 | 船體 | 網域 | 幾何 | 像素 | 計算 |
|--------|------|--------|----------|-------|---------|
|        | x    | x      |          |       |         |
