# <a name="sv_stencilref"></a>SV \_ StencilRef

SV \_ StencilRef 代表目前的圖元著色器的參考值。

## <a name="type"></a>類型



|      |
|------|
| 類型 |
| uint |



 

## <a name="remarks"></a>備註

您可以在 [D3D 11.3] 和 [D3D12] 中指定圖元著色器中的著色器參考值。 請參閱：

-   [D3D 11.3 著色器指定的樣板參考值](/windows/desktop/direct3d11/shader-specified-stencil-reference-value)
-   [D3D12 著色器指定的樣板參考值](/windows/desktop/direct3d12/shader-specified-stencil-reference-value)
