# <a name="f32tof16-function"></a>f32tof16 函式

將輸入轉換成 float16 型別。

## <a name="syntax"></a>語法

``` syntax
uint f32tof16(
  in float value
);
```

## <a name="parameters"></a>參數

<dl> <dt>

*值* \[在\]
</dt> <dd>

類型： **float**

輸入值。

</dd> </dl>

## <a name="return-value"></a>傳回值

類型： **uint**

轉換的值，儲存在 uint 的下半部。

## <a name="remarks"></a>備註

您也可以使用下列多載版本：

``` syntax
uint2 f32tof16(float2 value);
uint3 f32tof16(float3 value);
uint4 f32tof16(float4 value);
```

### <a name="minimum-shader-model"></a>最小著色器模型

下列著色器模型支援此函數。



| 著色器模型                                                        | 支援 |
|---------------------------------------------------------------------|-----------|
| [著色器模型 4](dx-graphics-hlsl-sm4.md) 和更高的著色器模型 | 是       |



 

雖然此函式適用于 [著色器模型 4](dx-graphics-hlsl-sm4.md) 和更新版本的著色器模型中，因為它是在4.0 和4.1 中進行模擬，所以在這些著色器模型上的 perfomant 比在 [著色器模型 5](d3d11-graphics-reference-sm5.md)上的還低。

下列著色器類型支援此函數：



| 頂點 | 船體 | 網域 | 幾何 | 像素 | 計算 |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
