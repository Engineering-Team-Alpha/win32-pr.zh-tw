# <a name="reserved-words"></a>保留字

下列單字保留給 HLSL 語言使用。 請勿使用它們來命名您 HLSL 程式碼中的變數或函數。



|                                     |
|-------------------------------------|
| 自動                                |
| 案例，catch                         |
| char、class、const \_ cast            |
| default、delete、dynamic \_ cast      |
| 列舉                                |
| explicit                            |
| friend                              |
| goto                                |
| long                                |
| mutable                             |
| new                                 |
| 運算子                            |
| 私用、受保護、公用          |
| 重新解譯 \_ cast                   |
| short、帶正負號、sizeof、靜態 \_ 轉換 |
| 範本，這會擲回               |
| try、typename                       |
| union、未簽署                     |
| using                               |
| virtual                             |
