# <a name="isinf"></a>isinf

判斷指定的值是否為無限。



| *ret* isinf (*x*)  |
|------------------|



 

## <a name="parameters"></a>參數



| 項目                                                   | 描述                            |
|--------------------------------------------------------|----------------------------------------|
| <span id="x"></span><span id="X"></span>*X*<br/> | \[在 \] 指定的值中。<br/> |



 

## <a name="return-value"></a>傳回值

傳回與輸入相同大小的值，如果 *x* 參數是 + INF 或-inf，則值設定為 **True** 。 否則 **為 False**。
