# <a name="texture2dmsarray"></a>Texture2DMSArray

Texture2DMSArray 類型 (存在於著色器模型 4) 加上資源變數中。



| 方法                                                                             | 描述                                        |
|------------------------------------------------------------------------------------|----------------------------------------------------|
| [**GetDimensions**](sm5-object-texture2dmsarray-getdimensions.md)                  | 取得資源維度。                      |
| [**GetSamplePosition**](sm5-object-texture2dmsarray-getsampleposition.md)          | 取得指定之範例的位置。         |
| [**載入**](texture2dmsarray-load.md)                                               | 取得一個值。                                    |
