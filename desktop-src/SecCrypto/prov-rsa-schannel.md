# <a name="prov_rsa_schannel"></a>>PROV \_ RSA \_ SCHANNEL

>PROV \_ rsa \_ schannel 提供者類型支援 Rsa 和 [*SCHANNEL*](../secgloss/s-gly.md) 通訊協定。

## <a name="algorithms-supported"></a>支援的演算法

如需這些演算法的說明，請參閱詞彙。



| 目的      | 支援的演算法                                                                                                                                                                                                                                                                                                          |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 金鑰交換 | [*RSA*](../secgloss/r-gly.md)                                                                                                                                                                                                                                                                   |
| 簽名    | RSA                                                                                                                                                                                                                                                                                                                           |
| 加密   | 任何 >PROV \_ RSA 安全通道 \_ CSP 都必須執行下列一或多個演算法： [ *RC4*](../secgloss/r-gly.md)<br/> [*DES*](../secgloss/d-gly.md)<br/> [*三重 DES*](../secgloss/t-gly.md)<br/> |
| 雜湊      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/>                                                                                                                                                                                             |
