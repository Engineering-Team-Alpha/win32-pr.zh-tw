# <a name="policy-object-protection"></a>原則物件保護

[**原則**](policy-object.md)物件預設會使用下列設定來保護：

-   本機群組世界具有一般 \_ 執行存取權。
-   已知的識別碼系統會授與原則 \_ 所有 \_ 存取權。
-   本機群組本機系統 \_ 管理員具有一般 \_ 讀取、一般 \_ 寫入和一般 \_ 執行存取權。
-   群組本機 \_ 系統管理員會被指派為此物件的擁有者和主要群組。

無法建立或終結 [**原則**](policy-object.md) 物件。
