# <a name="scestatus"></a>SCESTATUS

安全性設定工具集 API 會使用 **SCESTATUS** 資料類型來傳回函式呼叫結果的相關資訊。 如需有關已定義狀態碼值的詳細資訊，請參閱 [附件傳回值](management-return-values.md)。


```C++
typedef DWORD SCESTATUS;
```
