# <a name="iupdatedownloadcontentcollection-properties"></a>IUpdateDownloadContentCollection 屬性

**IUpdateDownloadContentCollection** 介面會定義下列屬性。



| 屬性                                                       | 描述                                                                                                                             |
|----------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get__newenum) | 取得用來列舉集合的 [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) 介面。                    |
| [**計數**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get_count)        | 取得集合中的項目數。                                                                                          |
| [**項目**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get_item)          | 從 [**IUpdateDownloadContentCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadcontentcollection) 介面取得更新的下載內容。 |
