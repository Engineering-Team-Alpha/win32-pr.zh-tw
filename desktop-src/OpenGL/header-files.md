# <a name="header-files"></a>標頭檔

使用的應用程式：

-   核心 OpenGL 函數必須包含標頭檔 <GL \\ gl>。
-   OpenGL 公用程式程式庫必須包含 <GL \\ x.glu 隊 .h> 的標頭檔。
-   OpenGL 程式設計指南輔助程式庫必須包含 <GL \\ glaux .h> 的標頭檔。
-   WGL 函式必須包含標頭檔的標頭檔。
-   支援 Microsoft 在 Windows 中執行 OpenGL 的 Windows 函式必須包含標頭檔案的 Windows .h。
