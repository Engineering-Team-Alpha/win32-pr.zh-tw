# <a name="compiling-the-sample-service-provider-using-nmake"></a>使用 NMAKE 編譯範例服務提供者

若要使用 NMAKE 編譯範例服務提供者，請開啟 Visual Studio 命令視窗，流覽至 <*SDK 安裝路徑* > \\ WMFSDK95 \\ WMDM \\ mdsp，然後 `nmake` 在提示字元中輸入。
