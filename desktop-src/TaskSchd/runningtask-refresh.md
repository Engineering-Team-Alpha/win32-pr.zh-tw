# <a name="runningtaskrefresh-method"></a>RunningTask 方法

針對腳本，重新整理工作的所有本機執行個體變數。

## <a name="syntax"></a>語法


```VB
RunningTask.Refresh()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
