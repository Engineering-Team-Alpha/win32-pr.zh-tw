# <a name="d1247-dynamic-copy-from-memory"></a>D1247：從記憶體進行動態複製

CopyFromMemory 無法將記憶體複製到從動態材質建立的點陣圖。



|             |         |
|-------------|---------|
| 錯誤層級 | 警告 |



 

## <a name="possible-causes"></a>可能的原因

CopyFromMemory 是在建立為 D3D11 使用動態的點陣圖上 \_ 呼叫 \_ 。

## <a name="possible-fixes"></a>可能的修正

請改用對應和取消對應來取代這種類型的點陣圖。
