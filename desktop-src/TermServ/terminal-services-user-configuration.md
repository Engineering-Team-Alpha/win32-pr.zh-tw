# <a name="remote-desktop-services-user-configuration"></a>遠端桌面服務使用者設定

[**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga)和 [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga)函數可讓您設定和取得指定使用者帳戶的設定資訊。 [**WTS \_ CONFIG \_ 類別**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class)列舉型別會定義使用者設定參數。
