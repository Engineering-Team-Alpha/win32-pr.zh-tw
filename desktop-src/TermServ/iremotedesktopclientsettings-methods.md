# <a name="iremotedesktopclientsettings-methods"></a>IRemoteDesktopClientSettings 方法

[**IRemoteDesktopClientSettings**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientsettings)介面會公開下列方法。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**ApplySettings 方法**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-applysettings)
</dt> <dd>

將指定的內容儲存在 RDP 檔案中。

</dd> <dt>

[**GetRdpProperty 方法**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-getrdpproperty)
</dt> <dd>

抓取單一名為 RDP 屬性值。

</dd> <dt>

[**RetrieveSettings 方法**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-retrievesettings)
</dt> <dd>

以字串形式抓取整個 RDP 檔案。

</dd> <dt>

[**SetRdpProperty 方法**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-setrdpproperty)
</dt> <dd>

設定單一名為 RDP 屬性的值。

</dd> </dl>
